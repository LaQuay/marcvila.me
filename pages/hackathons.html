A hackathon (hacking marathon, hack day, hack/codefest, or datathon) is a design sprint-like event; in which computer programmers, graphic designers, interface designers, domain experts, and others collaborate intensively on computer science projects.

The goal is to create functioning software or hardware by the end of the event. Hackathons tend to have a specific focus, which can include the programming language used, the operating system, an API or framework to be tested, ... . Although in some cases there are no restrictions.
